Workshop Details

Week 5 Workshop Task

In the week 5 workshop task, the git and github is revised. Students are supposed to make repository, and practice the git commands like clone, add, commit, push etc.

Students are given a set of questions that they need to solve. At each step of the answers, they are supposed to take screenshots of their task and include it in the word file.

This workshop task enables students to get familiar with git commands and how github works. They will be using git and github in their coursework task as well.
